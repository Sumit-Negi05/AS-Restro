## 📌 Overview

AS Restro is a desktop food ordering system built with Java Swing. It features a dual-panel design — a customer-facing ordering interface and a password-protected admin dashboard — with real-time cart management, automatic billing with discounts and GST, and a live sales log.

---

## ✨ Features

<table>
  <tr>
    <td width="50%" valign="top">

### 🧑‍🍳 Customer Panel
<ul>
  <li>Browse a menu of <strong>15 food items</strong></li>
  <li>Add items to cart with live total updates</li>
  <li>Auto-apply <strong>10% discount</strong> on orders above ₹1000</li>
  <li>GST (5%) calculated after discount</li>
  <li>Generate itemized bill with Order ID</li>
  <li>Cart clears automatically after order</li>
</ul>

</td>
<td width="50%" valign="top">

### 🔐 Admin Panel
<ul>
  <li>Secure login with credentials</li>
  <li>View <strong>total orders</strong> placed</li>
  <li>View <strong>total revenue</strong> earned</li>
  <li>Complete sales log with order details</li>
  <li>Refresh data dynamically at any time</li>
</ul>

</td>
</tr>
</table>
